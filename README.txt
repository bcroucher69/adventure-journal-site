AD's Adventure Journal - Local Website Package

WHAT'S INCLUDED
- Home page
- Journal page with saved text entries
- Photo gallery with image upload
- Voice journaling using your microphone
- Modern responsive design

HOW TO RUN
1. Unzip the package
2. Open index.html in a browser
3. Browse the pages from the top menu

IMPORTANT NOTES
- Data is stored in your browser's localStorage
- If you clear browser storage, the entries/photos/voice notes may be removed
- For voice notes, your browser will ask for microphone permission
- Smaller photos work best because the browser stores them locally

HOSTING LATER
You can host this easily on:
- Netlify
- Vercel
- GitHub Pages
